## 📝 第一步：写作动画剧本

Pika的剧本模块与ChatGPT深度集成。在Pika工作区中，你可以：

1. 打开"Script"面板
2. 输入故事梗概（如：一只想飞的小企鹅）
3. AI自动生成完整分镜脚本
4. 手动调整每一镜的文字描述

**优秀动画剧本三要素：**

| 要素 | 说明 | 示例 |
|:---|:---|:---|
| 情感弧线 | 角色从A点到B点的情感变化 | 害怕→勇敢 |

## 🎬 第二步：生成单镜头

## 🔗 第三步：组合多镜头叙事

Pika 3.0的"Scene Builder"是真正的杀手锏。它允许你将多个镜头串联成一个完整的故事，并保持角色一致性。

操作流程：
1. 在Scene Builder中添加新场景
2. 上传或生成角色参考图（Character Sheet）
3. 为该场景输入Prompt
4. Pika自动继承上镜头的角色外观
5. 调整转场效果（淡入淡出/硬切/叠化）

**一致性秘诀：** 在第一个镜头中生成角色的正面、侧面、3/4侧面三个参考图，保存为"Character Reference"。后续所有镜头引用该角色卡即可。

## 🎤 第四步：配音与口型同步

Pika 3.0的Lip Sync功能支持直接导入音频文件并自动匹配口型。步骤：

1. 在ElevenLabs或Fish Audio中生成配音
2. 导出MP3文件（建议48kHz）
3. 在Pika中导入音频，选择需要匹配的角色
4. 点击"Sync"等待处理（约1-2分钟）
