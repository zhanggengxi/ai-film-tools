## 🎯 技巧一：善用Synthesia的快捷键

大多数人只用鼠标操作，效率至少慢30%。记住这些快捷键：

- **Ctrl/Cmd + Enter** -- 快速预览
- **Ctrl/Cmd + S** -- 随时保存（养成习惯！）
- **Ctrl/Cmd + Shift + E** -- 直接导出

熟练后，你的创作速度能提升一倍。

## 🎯 技巧四：AI提示词公式

写AI提示词记住这个公式：**主体 + 动作 + 场景 + 风格 + 画质**

❌ 错误："一个女孩走路"
✅ 正确："一位年轻女性在东京雨夜的街道上漫步，赛博朋克风格，4K超高清，电影级光影"

越详细，AI给你的越好。
